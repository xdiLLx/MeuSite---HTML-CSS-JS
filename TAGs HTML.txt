<hgroup> Criar container para titulos(h1,h2,h3...)
<header> Definir Cabeçalho

<nav> - Contaniner para colocar o menu de navegação
<table> - Serve para definir uma tabela
<caption>(em tables) - Define o Titulo da tabela
<tr> - Define uma linha da tabela
<td> - define as cedulas das linhas - (rowspan - Total de linhas que a cedula vai ocupar)(colspan - Total de Colunas que vai ocupar)
<span> - Serve para vocÊ filtrar informação dentro de um texto
<figcaption> - Criar Legenda para fotos
<figure> Cria um container para a imagem, tornando mais facil mexer no css
<figcaption : hover> - Mostra uma formatação especifica quando o mouse estiver em cima da imagem
<map name=> serve para criar um mapa na imagem
<img usemap= #>serve para selecionar qual mapa vai ser utilizado
<area> - obrigatorio em maps, e sempre acompanhado da propriedade shape que defini o tipo de area, e coords, que define as coordenadas

//CSS
hover:Mostra uma formatação especifica quando o mouse estiver em cima
padding: espaçamento interno
margin: espaçamento externo
transition: temporizador para executar uma ação
position : Se relative vai movimentar somente no container, se absolute vai ser na pagina toda
text-transform : deixar o texto ou tudo minusculo ou maiusculo
display(em listas) : como vai ficar posicionado os textos, lado a lado ou um a baixo do outro
box-shadow- cria uma sombra em volta do objeto
float(em section, aside) : define os lados que cada container ficara
clear: both(obrigatório em footer ) : ajusta a pagina caso o float bugue
border-spacing(em tabelas) - Serve para tirar ou colocar espaço entre as celulas
margin-left - se definir como (auto) o navegador centraliza automatico
margin-right - se definir como (auto) o navegador centraliza automatico
text-indent - Espaço na primeira linha do paragrafo
opacity - Define opacidade do objeto
overflow(Em iframes) - Serve para impedir a rolagem do documento
::-webkit-scrollbar - seleciona a barra de scroll do documento
